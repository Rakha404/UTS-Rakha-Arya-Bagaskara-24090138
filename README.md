- Ringkasan Proyek

Proyek ini dibuat untuk mensimulasikan proses login dan tampilan dashboard admin sederhana dengan beberapa fitur utama sebagai berikut:

1. Halaman Login (index.html)

- Menyediakan formulir untuk masuk menggunakan Email dan Password.
- Email harus dituliskan dalam format yang valid (mengandung “@” dan karakter sesudahnya), sedangkan password bebas.
- Jika proses login berhasil, pengguna otomatis dialihkan ke halaman Dashboard.

2. Halaman Dashboard (dashboard.html)

- Menampilkan informasi ringkas seperti jumlah produk, total penjualan, dan total pendapatan.
- Seluruh data ditampilkan menggunakan objek JavaScript.
- Tersedia sidebar untuk navigasi antarhalaman.
- Ada tombol "Lihat Data Produk" untuk menuju halaman daftar produk.

3. Halaman Produk (products.html)

- Menampilkan daftar produk dalam bentuk tabel berisi kolom Nomor, Nama Produk, Harga, Stok, dan Aksi.
- Pengguna dapat mengedit informasi produk (menampilkan form edit) atau menghapus baris produk langsung dari tabel.
